# BunkerCore-V2 Code of Conduct

The use of BunkerCore-V2 implies full acceptance of the following digital coexistence and security standards:

## 1. System Integrity
Reverse engineering, code injection, Denial of Service (DDoS) attacks, or any attempt to violate the system's encryption protocols is strictly prohibited.

## 2. Prohibited Use
- Use of the system for illicit activities is strictly prohibited.
- The use of bots or automated scripts to exploit API endpoints outside of authorized authentication flows is prohibited.

## 3. Consequences
Any violation of this code will result in:
- Immediate and irreversible account blocking.
- Reporting to competent authorities if the activity constitutes cybercrime.
- Suspension of access to BunkerCore-V2 services.

System security is our absolute priority.
